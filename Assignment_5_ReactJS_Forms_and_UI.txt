Create a form in ReactJS that allows users to input their personal information, such as name, email,password,age,gender,address. The form should include a submit button that, when clicked, displays a message containing the user's inputted information. 
Instructions: 
	Create a new ReactJS project using create-react-app. 
	Create a new component called "PersonalInfoForm" that will contain the form. 
	Within the PersonalInfoForm component, create a form that includes input fields for name, email, and age. Each input field should have a label associated with it. 
	Add a submit button to the form that, when clicked, will display a message containing the user's inputted information. 
	Use the useState hook to manage the state of the form fields and the submit button.
